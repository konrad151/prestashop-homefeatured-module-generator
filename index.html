Homefeatured module generator

Generate module which adds additional tab on your homepage , set the module to see or not by client groups, choose from what category products will appear on your homepage tab

Module name must be only letters and no spaces

What are advantages?

How much you want

Generate how much modules you want, after install they will appear on your homepage with the specified before Tab name.

Groups visibility

If you want to see the module only by registered users or other special groups, you can do this in client group settings

Product display

Set in each module:

  • product quantity to display
  • category number which products will be from
  • random display
